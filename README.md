# LoginAndRun
Add's ability to run commands on login

Commands
==============

  - /Loginandrun (you can use /lar alias too)
  - /lar add (id;all)
  - /lar remove (id)
  - /lar list
  - /lar enable (id;all)
  - /lar disable (id;all)

Permission
==============

loginandrun.manager (users with this permission can manage their login commands)

Others
